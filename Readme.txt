# E2ECypressTesting_Cross-browserTesting_DebtPayOffPlaner
1.Download zip file and unzip it
2.Type cmd in Windows search
3.Give path in cmd of the project (eg:cd C:\Users\osama\Desktop\Debt_PayoffPlanner)
4.npm install
5.npx cypress open 
6.Cypress will open in a new window. 
7.Do the following i.e. Select E2E testing 
8.Select a browser and click "Start E2E testing"
9.This will open a new window and you should see a list of tests.
10.Double click on the test you want to run.
